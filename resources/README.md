Эта папка содержит ресурсы, которые представляют собой что-то кроме исполняемого кода, но требуются какими-то частями
исполняемого кода. При сборке эта папка кладётся рядом со скомпилированным исполняемым файлом.

В частности тут лежат задачи линейного программирования для тестирования скорости и корректности солверов. Задачи взяты
из https://github.com/YimingYAN/LP-Test-Problems/tree/master
