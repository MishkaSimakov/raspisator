# Модель задачи

Здесь описана модель задачи. Она взята из статьи "LP-based heuristics for scheduling chemical batch
processes" (F. Blomer et al).

Основная идея такова: Есть множество машин (Unit), каждая из которых в один момент времени может выполнять одну из
задач (Task). Отношение между ними - много-ко-многим. Каждая задача получает на вход какие-то продукты и производит
какие-то продукты. Задачи соединены с хранилищами (State), а хранилища соединены с задачами.
